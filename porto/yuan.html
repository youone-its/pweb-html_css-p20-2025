Projects

Robot Arm 5-DOF

Robot arm yang memiliki Degree of Freedom sebanyak 5 dimensi, dimulai dari bawah ke atas yaitu yaw-pitch-pitch-pitch-roll, menggunakan Invers Kinematik sederhana untuk pergerakannya, berbasis stm, DC motor dan potensio dari elektrikal, dan bahan besi, sehingga berat total robot mencapai lebih dari 20 Kg, digunakan untuk berbagai macam hal, biasanya untuk pick and place, atau sekadar robot edukasi

Robodog Fiammetta

Fiammetta merupakan quadruped, yaitu robot yang memiliki konfigurasi kaki 4 dengan model anjing (robodog), pengerjaan fiammetta menggunakan proses reinforcement learning yang masih berlangsung hingga saat ini, untuk menentukan pose dan aksi reaksi dari sensor (IMU, Lidar, dan feedback dari motor) ke masing masing aktuator di joint coxa femur tibia, lalu di masing masing motor juga direinforcement learning melalui ANN custom agar antara simulasi dan real-world memiliki aksi yang semirip mungkin

Game Interaktif untuk Anak Berkebutuhan Khusus

Membuat Game interaktif menggunakan scratch + opencv dan mediapipe untuk membuat media lain (papan tulis / tembok) sebagai "layar" permainan, dan memberikan edukasi pada para guru agar dapat membuat game atau sekadar memainkan game menggunakan yang sudah tersedia di platform scratch menggunakan opencv+mediapipe sebelumnya

Mobile Apps Kebun Raya Balikpapan

Sebuah aplikasi mobile untuk menunjang edukasi di Kebun Raya Balikpapan, memiliki fitur AR, 360 map virtual tour, Ensiklopedia, dan Quiz, menggunakan unity game engine untuk base nya (sebelumnya menggunakan flutter, tapi karna eror dependencies terkait AR, jadinya pindahin Unity)
